# 修心 Xiuxin 🧘

> 安住当下 · 修心修身

融合儒释道三家智慧的修心应用，帮助用户在忙碌的现代生活中建立内心的平静与力量。

**在线演示：** [http://bot.xjbcode.fun:8082/](http://bot.xjbcode.fun:8082/)

## ✨ 核心功能

### 🧘 呼吸练习（数息观）
- 可视化呼吸动画引导
- 自动计数 1-10 循环
- 开始/暂停控制

### 🪑 冥想计时器
- 4 种时长选择（5/10/20/30 分钟）
- 3 种法门切换（数息观/守窍/静坐）
- 倒计时显示与完成提示

### 📜 戒律打卡
- 5 条戒律每日功课
- 温和语言设计（"觉察"而非"破戒"）
- 现代诠释（如"不饮酒"包括不过度沉迷手机）

### 📊 修习统计
- 修习天数
- 累计分钟
- 连续天数
- 本地存储（localStorage）

### 📖 每日加持语
- 随机显示儒释道经典
- 来源：金刚经、道德经、论语、心经等

## 🎨 设计理念

1. **温和不评判** - 用"觉察"代替"破戒"，用"修习"代替"念佛"
2. **开放包容** - 三家智慧并存，用户自由选择
3. **隐私优先** - 本地存储，无需登录
4. **简约专注** - 核心功能不超过 3 次点击

## 🚀 快速开始

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/zhangfuwen/xiuxin-app.git
cd xiuxin-app

# 使用 Python 启动本地服务器
python3 -m http.server 8082

# 或使用 Node.js
npx serve .
```

然后在浏览器打开：http://localhost:8082/

### 部署

任何静态网站托管服务都可以：
- GitHub Pages
- Vercel
- Netlify
- Cloudflare Pages
